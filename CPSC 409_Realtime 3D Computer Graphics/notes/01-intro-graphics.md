# Introduction to Computer Graphics

## Tools for this course
- **OpenGL**: "Open Graphics Library"
    - A langauge-agnostic, cross-platform API for rendering on the GPU
    - Other competing libraries include Direct3D, Vulkan, Metal
- **SDL**: "Simple Directmedia Layer"
    - C API for creating windows, handling input, managing graphics
- **D**: a systems programming language
    - Compaitible with C libraries
    - Garbage collector!
    - Fun features like currying
- **GLM**
    - Math library for complex operations
